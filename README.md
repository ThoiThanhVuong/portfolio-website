# Personal Portfolio Website (CV Cá Nhân) - Thới Thanh Vương

Dự án Website giới thiệu năng lực và CV cá nhân dành cho vị trí tuyển dụng **Frontend Developer Intern** tại **GENX PK STORY**. Website được xây dựng trên nền tảng React.js hiện đại, tối ưu SEO, khả năng tương thích cao (Responsive) và giao diện Dark/Light mode ấn tượng.

## 🛠️ Công Nghệ Sử Dụng (Tech Stack)

- **Frontend Core**: React 19, JavaScript (ES6+), HTML5, CSS3 Variables.
- **Build Tool**: Vite (v6.0.0) tối ưu tốc độ biên dịch.
- **Routing**: React Router DOM (v7+) quản lý chuyển trang và trạng thái Active Navbar.
- **Animations & Effects**: Framer Motion (v12) hỗ trợ chuyển cảnh mượt mà giữa các trang và các vi tương tác (micro-interactions).
- **Icons**: Lucide React & FontAwesome React.
- **Effects**: Canvas Confetti tạo hiệu ứng pháo hoa khi gửi liên hệ thành công.
